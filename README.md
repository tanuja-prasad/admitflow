# AdmitFlow

Admission Lead Management Platform for Education Consultancies

## Problem Statement

Many education consultancies manage student leads using Excel sheets, WhatsApp chats, and manual follow-ups. This results in duplicate leads, missed follow-ups, poor visibility into counselor performance, and inefficient admission tracking.

AdmitFlow is a web-based platform designed to digitize and streamline the complete admission lead lifecycle—from inquiry to enrollment.
